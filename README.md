# Getting Agents to Give Up Their Secrets

A 50-slide presentation on AI best practices — from confused to in control.

## Structure

- **Narrative** — [docs/plans/2026-02-17-ai-best-practices-narrative.md](docs/plans/2026-02-17-ai-best-practices-narrative.md)
- **Deck plan** — [docs/plans/2026-02-18-ai-best-practices-deck-plan.md](docs/plans/2026-02-18-ai-best-practices-deck-plan.md)
- **Generator** — [generate-deck.js](generate-deck.js) (PptxGenJS + react-icons)
- **Output** — `Getting-Agents-to-Give-Up-Their-Secrets.pptx`

## Regenerating the deck

```bash
npm install
node generate-deck.js
```

Requires Node.js 18+.
