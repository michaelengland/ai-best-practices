# Workshop: Strategist → Operator — Narrative

> **Next step:** Use **/deckwright:craft** to structure this narrative into a content outline.

## Overview

- **Audience**: Attendees who completed Workshops 1-2. They prompt well and have custom instructions set up. They haven't used AI agents — they still copy-paste everything manually.
- **Purpose**: By the end, attendees have run real agent tasks where ChatGPT retrieved its own context and worked with their files and the web, and they understand when to trust and when to verify.
- **Key message**: Stop being the bottleneck. Let the agent find what it needs — you just review.
- **Framework**: Show → Do → Compare → Trust
- **Tone**: "Today the AI does the work. Your job is to judge."
- **Format**: ~25 slides, ~45 min effective. Billboard design. **ChatGPT Org** — everyone already has this.

## Tool Requirements

This workshop uses **ChatGPT Org**, which all attendees have access to. No additional setup or accounts required.

ChatGPT Org natively provides everything needed:
- **File upload**: Read and process documents, spreadsheets, PDFs
- **Web browsing**: Search and summarise live web content
- **Code interpreter**: Analyse data, run calculations
- **Privacy**: Org conversations are not used to train ChatGPT — safe for company data

The facilitator should prepare 2–3 sample documents before the session (a policy doc, a data file, a memo). Attendees can also use their own files.

## Session Structure

### Opening: Where We're At (3–5 min)

- Show the spectrum. Highlight Strategist → Operator crossing.
- "You talk to AI like a pro. You've set up its playbook. But who's doing all the work? Still you."
- Quick demo: show the 4-step copy-paste trap (You search → You paste → AI drafts → You apply). "Every step has 'you' in it."

### The Ferrari Metaphor (1 min)

- "You're using a Ferrari to carry groceries."
- Let it land. This should sting — they recognise themselves.

### Exercise 1: Your First Agent Task (10 min)

- **Teach** (3 min): Show the agent-driven workflow: Agent retrieves → Agent identifies gaps → Agent creates → You review. "Same four steps. You only appear at step 4."
- **Setup**: Open ChatGPT. Upload a document using the paperclip icon (or drag and drop).
- **Exercise**: Give ChatGPT a task: "Read the document I've uploaded and write a one-paragraph summary for a non-technical audience."
- **Watch**: ChatGPT reads the file, processes it, and produces output — no copy-paste from you.
- **Discuss**: "How long would that have taken you manually? ChatGPT just did it in seconds."

### Exercise 2: Add Connections (10 min)

- **Teach** (2 min): "ChatGPT can reach beyond what you hand it." Show the native tool capabilities: Web search, File upload, Code interpreter, Image generation.
- **Exercise**: Give ChatGPT a research task with web browsing: "Research [topic] and summarise the top 3 findings with sources." (Web browsing is on by default in ChatGPT Org.)
- **The reveal**: ChatGPT found information you didn't think to provide. It went looking for what it needed.
- **Compare**: "Without this, you'd have searched, read, copy-pasted, and summarised yourself. ChatGPT just did all four."

### Exercise 3: The Budget Memo Moment (8 min)

- **Teach** (1 min): The budget memo story from the main deck — "You forgot the budget memo. The VP asks about it. Too late."
- **Exercise**: Give ChatGPT a more complex task — upload multiple files or combine a file upload with a web search. Something where relevant context isn't obvious.
- **Watch for**: Did ChatGPT surface something you wouldn't have thought to include?
- **Discuss**: "The agent found context you wouldn't have thought to provide. That's the power of giving it access."

### Trust, But Verify (5 min)

- **Teach** (2 min): "You just handed AI the keys. How do you know it's right?"
  - Never trust numbers you didn't provide
  - Verify claims against sources
  - Use AI for drafts, not decisions
  - Note: ChatGPT Org doesn't train on your data — but that's about privacy, not accuracy
- **Exercise**: Take the output from Exercise 2 or 3. Pick one specific claim or number. Verify it manually.
- **Discuss**: "Was it accurate? Did anything feel right but turn out to be wrong? That 'feels right' moment is the most dangerous."

### Closing: What's Next (3 min)

- "You're now an Operator. You set up agents, connected them to your world, and let them work."
- **The cliffhanger**: "One agent doing one task is powerful. But real work has multiple steps. What if you had a whole team — each one specialised?"
- "Next time: Operator → Orchestrator. We chain agents together. Each one focused. The team does everything."

## Notes

- **No tool setup needed**: Everyone has ChatGPT Org. The facilitator's job before this session is preparing sample documents, not troubleshooting installs.
- **File preparation**: Have 2–3 sample documents ready (a policy doc, a data file, a memo). Keep them generic enough for the group task but specific enough to be interesting. Attendees can also use their own files — encourage this as it makes the exercise more personally relevant.
- **Privacy reassurance**: Mention early that ChatGPT Org doesn't use their conversations for training. This matters for people hesitant to upload company documents.
- **Trust exercise timing**: If short on time, the trust exercise can be a group discussion instead of individual verification — but don't skip it. It's the most important mindset shift in the series.
- **Teaser for Workshop 4**: End with a specific hook — "Next session, each agent gets its own specialisation. You'll see why that matters."
